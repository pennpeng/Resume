    • 苏州工业园区居民通和企业通项目

      负责居民通和企业通的平台及应用插件的部署、维护和更新工作，项目采用 WebApp 的形式，运行在基于 openshift origin 二次开发的 PaaS 平台之上，目前，平台已陆续实现对公积金、社保、环保、科技、终身教育、智慧社区、社会信用、劳保等部门应用和第三方增值应用系统的接入、服务和管理，在公众服务、政府服务、创新应用等方面产生预期成效

      后端数据存储使用了 MySQL、MongoDB 和 Redis，数据库均部署在虚拟机上，并通过 service 的方式被 Openshift 平台中的容器调用

      数据库高可用方面，MySQL 使用 MHA 来实现高可用，MHA 在监控到 master 节点故障时，会提升其中拥有的最新数据的 slave 节点成为新的 master 节点；MongoDB 通过副本集提供其服务的高可用及主从复制，然而当服务发生故障，从服务切换到主服务时，相应的 IP 或者端口会发生变化，为了保证 MongoDB 对外提供的服务接口不变，引入了LVS

      应用的元数据存储在 ETCD 上，ETCD 采用集群以保证平台的高可用；整个 OPS 平台的网络采用 openvswitch 的 vxlan 模式，通过 openflow 协议来处理数据流

      应用的代码 push 到本地的 git 仓库中，通过 openshift 的 imagestream 和 buildconfig 来自动构建镜像并启动容器，从而实现应用的自动部署和持续交付
